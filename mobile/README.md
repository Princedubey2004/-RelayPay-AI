# Prince Dubey AI - Mobile

The mobile client app for Prince Dubey AI, built with React Native and Expo.

## Features
- **Complete Interface**: Screens for Auth (Login/Register), Home, transactions (Payment), History tracking, and User Profile.
- **Offline Modes**: A dedicated queue system utilizing real-time network detection to securely queue transactions locally when offline and successfully push them to the backend upon network stabilization.

## Setup and installation
1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Expo Development Server:**
   ```bash
   npm start
   ```

Once the Expo CLI starts, you can:
- **Scan the QR code** via Expo Go on your mobile device.
- Press `a` for an Android Emulator.
- Press `i` for an iOS Simulator.
